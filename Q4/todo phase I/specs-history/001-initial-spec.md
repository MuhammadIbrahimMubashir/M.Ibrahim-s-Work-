# Todo App Phase I Specification

## Objective
Build a simple in-memory Todo CLI application.

## Features
1. Add a task with title and description
2. View all tasks
3. Update task title or description
4. Delete task by ID
5. Mark task as complete or incomplete

## Constraints
- No database
- No external libraries
- Simple Python only
